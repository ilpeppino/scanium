Scanium AI + Pricing Regression Checklist (5 min)

Device setup
- Use dev build with network access
- Ensure assistant is enabled in Settings

1) Scan -> hypotheses -> confirm -> item created
- Open Camera
- Capture a clear object
- Expect: Hypothesis sheet shows ranked options and "None of these"
- Tap a hypothesis
- Expect: item is created and appears in Items list
- Screenshot: hypothesis sheet with ranked options

2) Edit item -> AI chooser -> price path opens
- Open any item -> Edit
- Tap the AI (sparkles) button
- Expect: chooser sheet with "Price my item" and "Generate listing text"
- Tap "Price my item"
- Expect: Pricing Assistant sheet opens (not Export Assistant)
- Screenshot: AI chooser sheet and Pricing Assistant header

3) Price path -> calculate -> result or clear unavailable message
- In Pricing Assistant, enter missing fields if prompted
- Tap "Confirm and estimate"
- If pricing enabled: expect price range + sources list
- If pricing disabled: expect "Pricing not available" placeholder sheet
- Screenshot: pricing result with sources OR unavailable sheet
